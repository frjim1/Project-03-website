# Project Scope Statement
## Nova Coffee shop
## Project purpose
The purpose of this project is to create a fictional multi-page website for Nova
Coffee Shop. The website will establish an online presence for the bussiness,
showcase the menu items, and allow customers to easily find the coffee shop's
location and contact information.
## Feedback
Feedback will be provided by the instructor during project reviews and after
assignment submissions. Classmates will also provide feedback on the website's
design and functionality. The website will be tested to ensure all links work
properly and that each page functions as expected.
## Description
The project involves creating a multi-page website for Nova coffee shop using HTML
and CSS. The website will use a simple layout that is easy to navigate and features
a clean, professional design.
## People resources
* Prjoect Manager/Developer: Frances Ramirez
* Instructor
* Classmates
* Potential website visitors (customers)
## Desired Results
The completed website should:
* Display all required webpages.
* Include working navigation links.
* Maintain a clean and professional apperance.
* Be able to be responsive on different screen sizes.
* Meet all course requirements.
## Priorities
* Time
* Quality
* Budget
## Time
* Project planning: August 10-12, 2026
* Website developement: August 13-27, 2026
* Testing and revision: August 28-30, 2026
* Final submission: September 1, 2026
## Qaulity
The project will be considered complete when the website functions correctly, all
navigation links work properly, the pages display consistently, and all assignment
requirements have been met.
## Budget
There is no financial budget required. The website will be developed using free
software tools such as Visual Studio Code and GitHub.
## Exclusions
The following features are outside the project scope:
* Online ordering
* Customer login
* Payment processing
* Mobile application
* Employee scheduling
* Inventory management
## Acceptance Criteria
The project will be finalized once it has been approved and signed off by the
course instructor and all project requirements have been completed by the deadline.
