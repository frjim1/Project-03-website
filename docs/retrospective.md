# Project Retrospective
## What Went Well
I succefully created a multi-page website for Nova Coffee Shop using HTML and CSS. Throughout this project, I became more comfortable using Visual Studio Code, GitHub, and connecting multiple webpages with a shared stylesheet.
## What Went Wrong
At first, I had difficulty linking HTML and CSS files correctly and understanding how to organize my respository. I spent extra time fixing formatting and navigation issues. 
## What I Would Do Differently
If I completed this project again I would make more time planning before I start coding. Creating a clear outline first would help me stay organize and complete the project more efficiently.
