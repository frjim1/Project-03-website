# Project Plan
## Project Overview 
The goal of this project is to create a multi-page website for Nova Coffe Shop using HTML and CSS. The website will provide information about the coffee shop, including it's menu, location, and contact information. The website will be using GitHub for version control and Visual Studio Code for developement.
## TAME Risk Analysis
### Time 
**Risk** I may not have enough time to complete all of the project requirements before the deadline.
**Mitigation Strategy** I will work on the project a little each day and keep track of my progress so I finish on time. 
### Application 
**Risk** The website may contain coding errors or broken navigation links.
**Mitigation Strategy** I will test every webpage and make sure all links and features work correctly before submitting the project.
### Management 
**Risk** Keeping the project organzied and making sure all required files are completed before the deadline could be challenging.
**Mitigation Strategy** I will follow the project requirements, keep my files organized, and review the assignment checklist before submitting my work.
### Environment
**Risk** Computer problems or internet issues could delay my progress or cause me to lose work.
**Mitigation Strategy** I will save my work frequently, commit changes to the GitHub regularly, and keep backup copies of my project files.
